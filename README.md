# Welcome to the f25-cisc474-red wiki/ readme!

## Problem Statement & Motivation

* It is often very difficult to stay motivated to do your tasks, and it is very easy to get distracted and do something else. We aim to develop a web application that will help keep people motivated to complete their tasks while also making the process enjoyable.


## Objectives

### Objective of the Project 
* Make a game for users to be able to focus better on doing all of their tasks without putting them off by gamifying their chores! 

### High-level features (MVP)
* Users should be able to enter tasks/task lists 
* Users should be able to log in so tasks persist and save task sets
* Entered tasks should turn into fish
* Users should then be able to fish up these fish which will determine what task they have to complete next
* After fishing a task, the user should be able to mark it as completed
* A completed task will disappear from the pond
* More tasks can be added mid-game

## Optional features
* Premade lists of tasks
* Optional self-care or break fish
* Time limit
* Random different types of fish appear
* Fish pokedex


## Stakeholders and Intended Users
* People who lose track of what they’re doing
* People with adhd or choice paralysis
* People who need extra motivation to get stuff done
